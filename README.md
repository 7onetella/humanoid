# humanoid
Generic chatbot for slack

### how to build binary
go build -o humanoid

### how to run
```
export SLACK_BOT_USER_OAUTH_ACCESS_TOKEN=xxxxx; \
export SLACK_BOT_MEMBER_ID=XXXXXXX; \
export SLACK_BOT_DEBUG=true; \
./humanoid
```
